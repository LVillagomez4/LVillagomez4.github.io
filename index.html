Featured Project

Micro Track: Budgetting App

This is a full-stack web application that allows users to track product prices from online stores. Users can set a target price, and when the product drops below that value, they receive an email alert. The system supports both automated price updates through web scraping and manual database updates.

Tech Stack:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js (Express)
  • Database: MySQL
  • Other: Web scraping + Email notifications (Nodemailer)
